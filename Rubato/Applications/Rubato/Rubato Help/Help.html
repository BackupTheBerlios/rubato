Welcome to Rubato Help!

Anja Fleischer is currently working on an update of Rubato Help for Mac OS X. You might want to look for updated documentation or see the documentation of Rubato for NeXTStep at http://www.rubato.org.

For an overview about related work by Guerino Mazzola see http://www.encyclospace.org. For an overview about related work at the KIT-MaMuTh Research Group for Mathematical Music Theory (TU Berlin) see http://www.mamuth.de.

The Rubato source code is available under the GNU General Public License. If You are interested in the Rubato development, have a look at http://rubato.berlios.de.

This product includes F-SCRIPT, a software developed by Philippe Mougin. (See http://www.fscript.org.) The F-SCRIPT interpreter offers You a scripting interface to the programming classes and the inner state of Rubato. To do so properly, You must know the class structure and the application structure of Rubato (see http://rubato.berlios.de.). WARNING: THE IMPROPER USE OF THIS COMPONENT MAY CAUSE DAMAGE TO YOUR SYSTEM, SO BE CAREFUL.
